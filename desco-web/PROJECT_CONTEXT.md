# DESCO Web Project Context

## Overview
This project is the official web application for **DESCO ANTISESPTICS**, a company specializing in professional hygiene and infection control solutions. The platform showcases high-authority, medical-grade products compliant with ISO 13485:2016 and DRAP standards.

## Goals
- Serve as a digital product catalog and informational site for DESCO's medical and hygiene products.
- Provide categorized product collections (e.g., Hand & Skin, Surface & Floor, Instruments & Endoscopes).
- Display detailed efficacy data, microbiological activity, and technical specifications for medical professionals.

## Technologies
- **Framework:** Next.js 16 (React 19)
- **Language:** TypeScript
- **Styling:** Vanilla CSS / CSS Modules
- **Icons:** Lucide React

## Key Considerations
- **Static Data Management:** Data is managed statically within the codebase (`app/data.ts`) rather than via an external database or CMS.
- **Design System:** The UI follows a professional, medical aesthetic with a category-first hierarchy and distinct collection themes. Cleanliness and high contrast are prioritized.
